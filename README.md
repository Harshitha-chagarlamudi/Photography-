# Photograph-Website
A Dynamic photography web application using HTML, CSS GRID.
Use image tags to seamlessly integrate high-quality photographs.
Apply responsive design principles with CSS for an optimal viewing experience.
Design an aesthetically pleasing gallery with hover effects for a captivating showcase.
